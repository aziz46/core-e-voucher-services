# Documentation Directory

This directory contains API documentation and service-specific documentation.

## Structure

- `openapi/` - OpenAPI 3.0 specifications for each service API

## OpenAPI Specifications

Each service has its own OpenAPI specification file:
- `credit-service.yaml` - Credit Limit Service API
- `billing-service.yaml` - Billing Service API
- `ppob-core.yaml` - PPOB Core Service API

These specifications document:
- API endpoints
- Request/response schemas
- Authentication requirements
- Example requests and responses
- Error codes and messages

## Viewing the Docs

You can view the OpenAPI specs using tools like:
- Swagger UI
- Redoc
- Postman (import the YAML files)
